# Project Title (Update)

My project is a multipurpose clock app, it has a world clock, a timer, and a stopwatch.


## Instructions for Build and Use

Steps to build and/or run the software:

1. Go to the website
2. That's it

Instructions for using the software:

0. Each widget has it's own functionality -
1. World Clock - select the dropdown menu to select time zone
2. Timer - enter number of minutes, then press start
3. Stopwatch - press play, pause, or reset

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:
